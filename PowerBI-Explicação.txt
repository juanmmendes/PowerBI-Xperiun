# 🧱 Os 5 Pilares para Criar um Dashboard Poderoso

### 1. 📥 Extração de Dados

* **Onde estão os dados?** ERPs, planilhas, bancos de dados, APIs, etc.
* O **Power BI não insere dados**, ele apenas **lê dados de fontes conectadas**.
* Explore a etapa de **"Transformar Dados"** para visualizar e navegar nas alterações feitas.

---

### 2. 🔧 Transformação dos Dados (Power Query)

* **Limpeza e estruturação**: substituir valores, corrigir erros, eliminar inconsistências.
* **Unificação de tabelas**: combine colunas (ex: nome + sobrenome), formate datas (dia/mês/ano).
* **Definição do tipo de dado**: evite "ABC123" (tipo indefinido) – use **"Detectar Tipo de Dados"**.
* **Criação de colunas calculadas**: ex: `Receita = Valor * Quantidade`.
* **Referência cruzada** com tabelas auxiliares (ex: tipo de produtos).
* Finalize com **"Fechar e Aplicar"**.

---

### 3. 🔗 Modelagem dos Dados

* Relacione tabelas por chaves (ex: `ProdutoID`, `ClienteID`).
* Utilize **medidas DAX** para cálculos dinâmicos, como:

  ```DAX
  Custo Total = SUMX(Vendas, Vendas[Quantidade] * RELATED(Produtos[CustoUnitario]))
  ```
* Modelagem correta evita erros e **melhora a performance do relatório**.

---

### 4. 📊 Cálculos e Indicadores (DAX)

* Use a **linguagem DAX** para criar:

  * Custos, Receitas, Lucros, Margens
  * Indicadores por período, produto ou canal
* Funções úteis:

  * `RELATED()` – busca dados em tabelas relacionadas
  * `CALCULATE()`, `SUMX()`, `FILTER()`, entre outras
* Selecione o **melhor visual para cada indicador**

---

### 5. 🖼️ Visualização Estratégica

* Construa **painéis úteis, intuitivos e bonitos**.
* Use gráficos adequados (barras, pizza, linha, área) com design limpo.
* Configure eixos corretamente (ex: eixo Y começando do zero).
* Utilize **filtros interativos (CTRL + clique)** para insights mais profundos.
